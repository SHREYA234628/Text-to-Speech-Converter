# Text-to-Speech Converter

A simple and elegant web-based Text-to-Speech converter that allows you to convert any text into speech with customizable voice options.

## Features

- 🎤 Convert text to speech using Web Speech API
- 🎵 Multiple voice options
- ⚡ Adjustable speech rate (speed)
- 🎚️ Adjustable pitch
- ⏯️ Play, pause, resume, and stop controls
- 📱 Responsive design for mobile and desktop
- 🎨 Beautiful gradient UI

## How to Use

1. Open `index.html` in a modern web browser (Chrome, Edge, Safari, or Firefox)
2. Enter or paste your text in the text area
3. Select your preferred voice from the dropdown menu
4. Adjust the speed and pitch using the sliders
5. Click the "Play" button to start the speech
6. Use pause/resume/stop buttons to control playback

## Browser Compatibility

This application uses the Web Speech API, which is supported by:
- ✅ Google Chrome
- ✅ Microsoft Edge
- ✅ Safari
- ✅ Firefox (partial support)

## Files

- `index.html` - Main HTML structure
- `style.css` - Styling and responsive design
- `script.js` - JavaScript functionality using Web Speech API

## Technologies Used

- HTML5
- CSS3 (with gradients and flexbox)
